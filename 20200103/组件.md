### 组件的传递

传递:
    通过子组件的属性来传<div num="3"></div>(静态的)或者
    <div :num="pnum"></div>"(动态的)
    接shou:{
        props:['num']
    }


    在父级传递数据给自己的时候,可以让自己拥有父级的数据并且不与父级数据相关联
    1.通过子组件的属性来传
    <div :num="pnum"></div>(动态的)
    2.使用props接收
    {
        props:['num']
    }
    3.把接收过来的数据编程自己的
    {
                props:['num'],
                data(){
                    return {
                        cnum:this.num
                    }
                }
            }



子传父:
1.父级需要定义一个改变自己数据的方法
2.自己需要定义一个事件,去调用父级的方法
this.$emit('自定义时间名',可以传参)
3.在子组件的行间绑定自己的事件,值为父级的修改数据的方法
